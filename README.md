# Disquo

Data comparator for humans.

## Install

1. `git clone http://github.com:NoLiD/Disquo.git`
2. `npm install`
3. `bower install`
4. `npm install -g ember-cli`

## Build

`ember build`

## Run

1. `ember server`
2. Visit your Disquo at http://localhost:4200.
